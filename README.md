# Casos de Teste - Sistema de Login:

Este repositório contém casos de teste de diferentes sistemas desenvolvidos para validar o sistema de autenticação de usuários.

## Objetivo:

Garantir que o sistema:

- Permita acesso com credenciais válidas
- Bloqueie acesso com dados inválidos
- Exiba mensagens adequadas
